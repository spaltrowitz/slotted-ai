#### High Issues (6)

2. No future-time validation on meetup creation (past meetups accepted)
3. No endTime > startTime validation (backward meetups possible)

5. No CRUD endpoints for `friend_groups` table

#### Medium Issues (11)

- Upsert on re-request resets other user's friendship_type
- Group decline leaves meetup in limbo forever
- Calendar notification times use server locale, not recipient TZ
- Calendar sync delete-before-insert race window (zero availability)
- Webhook double-sync (full sync + events.list redundant)
- `meetup_request` type overloaded → false dedup
- No push suppression when app is in foreground

#### Low Issues (5)
