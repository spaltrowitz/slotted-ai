2. **Star rating localStorage-based** — Dismissed/rated meetup IDs ephemeral; if user clears storage, may see prompt again (acceptable)
3. **Social Battery defaults to "open" hidden** — Backend default `2-3-week` still applies to scheduling; UI just hidden until 3+ hangouts
